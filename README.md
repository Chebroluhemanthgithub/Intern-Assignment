Task Management web application 
It was built using React JS , mainly designed for 2 roles 
1) Project Manger
2) Users

Main features 
ProjectManaget features
Add Task  with Title, Description,Deadline ,Assigned User
Edit Task  
Delete Task
Update task status
Delete Task
Update task status
sell all tasks


User features 
login
see only tasks assigned to user
update status only : pending , Inprogress, Done 
No option to edit by user

Install

npm create vite@latest taskmanagementweb ----Template  react
