<h1>Task Management web application</h1>
<p>It was built using React JS , mainly designed for 2 roles</p>
1) Project Manger
2) Users

<h3>Main features </h3>
<h2>ProjectManaget features</h2>
Add Task  with Title, Description,Deadline ,Assigned User
Edit Task  
Delete Task
Update task status
Delete Task
Update task status
sell all tasks


<h2>User features </h2>
login
see only tasks assigned to user
update status only : pending , Inprogress, Done 
No option to edit by user

Install

npm create vite@latest taskmanagementweb ----Template  react
